Interview Questions · Machine Learning

Classic ML Interview Questions & Answers

Twenty foundational machine-learning questions covering theory, models, evaluation, and optimization — the staples of any ML interview round.

1 · Foundations

📐 Core Concepts

1 What is the difference between supervised and unsupervised learning?
Supervised learning uses labeled data (input–output pairs) to learn a mapping function (e.g., classification, regression). Unsupervised learning finds patterns in unlabeled data (e.g., clustering, dimensionality reduction).
2 Explain the bias–variance tradeoff.
Bias is error from overly simplistic assumptions (underfitting); variance is error from sensitivity to training data fluctuations (overfitting). Total error = bias² + variance + irreducible error. Increasing model complexity lowers bias but raises variance; the goal is to minimize total error.
3 What is overfitting and how do you prevent it?
Overfitting is when a model learns noise in training data and performs poorly on unseen data. Prevention: regularization (L1/L2), cross-validation, more training data, simpler models, dropout, early stopping, and pruning.
4 Difference between L1 and L2 regularization?
L1 (Lasso) adds the absolute value of weights to the loss, producing sparse solutions (feature selection). L2 (Ridge) adds the squared weights, shrinking them toward zero but rarely to exactly zero.
5 What is cross-validation?
A technique to evaluate model generalization by splitting data into k folds, training on k−1 folds and validating on the remaining one, repeated k times. K-fold (commonly k=5 or 10) reduces variance in performance estimates.
2 · Metrics, Trees & Optimization

📊 Evaluation & Learning Mechanics

6 Explain precision, recall, and F1-score.
Precision = TP / (TP + FP) — accuracy of positive predictions. Recall = TP / (TP + FN) — coverage of actual positives. F1 = harmonic mean = 2·(P·R) / (P + R) — balances both, useful for imbalanced classes.
7 What is the difference between bagging and boosting?
Bagging trains models in parallel on bootstrap samples and averages results to reduce variance (e.g., Random Forest). Boosting trains models sequentially, each correcting the previous one's errors, reducing bias (e.g., AdaBoost, Gradient Boosting).
8 How does a decision tree decide splits?
It selects splits that maximize information gain (reduce entropy) or minimize Gini impurity for classification, and minimize variance/MSE for regression.
9 What is the curse of dimensionality?
As dimensions increase, data becomes sparse, distances between points become less meaningful, and models need exponentially more data to generalize well. Mitigated via dimensionality reduction (PCA) and feature selection.
10 Explain how gradient descent works.
An optimization algorithm that iteratively updates parameters in the opposite direction of the loss gradient: θ = θ − α·∇J(θ), where α is the learning rate. Variants: batch, stochastic (SGD), and mini-batch.
3 · Models & Algorithms

🧠 Model Families

11 What's the difference between generative and discriminative models?
Generative models learn the joint probability P(x, y) and can generate data (e.g., Naive Bayes, GANs). Discriminative models learn the conditional P(y | x) or decision boundary directly (e.g., logistic regression, SVM).
12 How does logistic regression work?
It models the probability of a binary outcome using the sigmoid function applied to a linear combination of features. It's trained by minimizing log-loss (cross-entropy) and outputs probabilities mapped to classes via a threshold.
13 What is PCA?
Principal Component Analysis is a linear dimensionality reduction technique that projects data onto orthogonal axes (principal components) capturing maximum variance, found via eigendecomposition of the covariance matrix.
14 How does K-means clustering work?
It partitions data into k clusters by: (1) initializing centroids, (2) assigning points to nearest centroid, (3) recomputing centroids as cluster means, repeating until convergence. Sensitive to initialization and requires choosing k.
15 What is the difference between parametric and non-parametric models?
Parametric models assume a fixed form with a finite number of parameters (e.g., linear regression). Non-parametric models grow in complexity with data and make fewer assumptions (e.g., KNN, decision trees).
4 · Practical Considerations

⚙️ Real-World Practice

16 How do you handle imbalanced datasets?
Resampling (oversampling minority/SMOTE, undersampling majority), class weighting, using appropriate metrics (F1, ROC-AUC, PR-AUC instead of accuracy), and ensemble methods.
17 What is the ROC curve and AUC?
The ROC curve plots True Positive Rate vs. False Positive Rate across thresholds. AUC (Area Under Curve) measures overall ability to discriminate classes; 1.0 is perfect, 0.5 is random.
18 Explain the difference between bagging-based Random Forest and a single decision tree.
A single tree is prone to high variance/overfitting. Random Forest builds many trees on bootstrapped samples with random feature subsets and averages predictions, reducing variance and improving generalization.
19 What is regularization and why use it?
Regularization adds a penalty to the loss function to discourage complex models, reducing overfitting and improving generalization by constraining parameter magnitudes.
20 What's the difference between batch, mini-batch, and stochastic gradient descent?
Batch uses the entire dataset per update (stable, slow). SGD uses one sample (fast, noisy). Mini-batch uses a small subset (balances stability and speed) — the most common in practice.
