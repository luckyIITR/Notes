Training LMs to Follow Instructions with Human Feedback

Reasoning & Process Supervision
🔬

Process Reward Model · Reasoning · Step-Level Supervision

Let's Verify Step by Step

Instead of rewarding only the final answer (ORM), reward each reasoning step with a Process Reward Model (PRM) — enabling precise error localization and step-level search. Foundation of o1-style reasoning models.

🧮

Mathematical Reasoning · GRPO · Pretraining · RL

DeepSeekMath: Pushing the Limits of Mathematical Reasoning

Introduces GRPO — eliminates the PPO critic by using group-normalized rewards as the baseline. A 7B model matches GPT-4 on MATH. Directly inspired DeepSeek-R1.

🌟

Reasoning · GRPO · RL from Scratch · Distillation

DeepSeek-R1: Incentivizing Reasoning via RL

The first open-source model to match o1 — reasoning emerges from pure RL (R1-Zero) with no SFT. The full R1 uses a 4-stage pipeline with cold start, GRPO, rejection sampling, and distillation to smaller models.
